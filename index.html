HTML

Pendahuluan

  1. Apa itu HTML?

    2. HTML, singkatan dari Hyper Text Markup Language, merupakan bahasa markup standar yang digunakan untuk membangun struktur halaman web serta menggambarkan kontennya melalui sejumlah elemen. Elemen-elemen ini memberikan identifikasi pada berbagai bagian konten seperti judul, paragraf, tautan, dan lainnya, sambil memberikan petunjuk kepada browser mengenai cara menampilkan konten tersebut. Dengan menggunakan HTML, para pengembang web dapat membentuk tampilan halaman yang terstruktur dan memungkinkan pengguna untuk berinteraksi dengan berbagai elemen dan informasi.

  2. Contoh Dokumen Sederhana HTML

    3. <!DOCTYPE html>
    <html>
      <head>
        <title>Judul Halaman</title>
      </head>
      <body>
        <h1>Judul Pertama Saya</h1>
        <p>Paragraf pertama saya.</p>
      </body>
    </html>

    Deklarasi <!DOCTYPE html> digunakan untuk menetapkan bahwa dokumen ini adalah dokumen HTML5. Elemen <html> berperan sebagai elemen induk atau elemen root dari halaman HTML. Elemen <head> digunakan untuk menyimpan informasi meta tentang halaman. Elemen <title> mengatur judul yang akan muncul pada bilah judul browser atau tab halaman tersebut. Elemen <body> menandai area dokumen yang tampak pada layar, menjadi wadah untuk semua elemen konten yang terlihat, seperti judul, paragraf, gambar, tautan, tabel, daftar, dan lainnya. Selain itu, elemen <h1> dan <p> digunakan untuk menetapkan judul besar dan paragraf secara berurutan sesuai dengan struktur konten halaman.

CSS

Pendahuluan

  1. Apa itu CSS?

    2. CSS, atau Cascading Style Sheets, adalah bahasa pemrograman yang digunakan untuk mengatur tampilan elemen-elemen HTML di layar, kertas, atau media lainnya. Dengan CSS, kita dapat menghemat banyak pekerjaan karena dapat mengontrol tata letak dari beberapa halaman web sekaligus. Gaya-gaya eksternal disimpan dalam file CSS terpisah untuk memudahkan pengelolaan dan penggunaan ulang.

  2. Contoh Dokumen Sederhana HTML dengan style CSS

    3. <!DOCTYPE html>
    <html>
      <head>
      <style>
        p {
          text-align: center;
          color: red;
        }
      </style>
        <title>Judul Halaman</title>
      </head>
      <body>
        <h1>Judul Pertama Saya</h1>
        <p>Paragraf pertama saya.</p>
      </body>
    </html>

    Dalam aturan CSS di atas dalam blok elemen <style>, p adalah selector yang mengacu pada semua elemen paragraf dalam dokumen HTML. Kemudian, dalam blok deklarasi, kita menentukan bahwa properti text-align diatur ke center untuk membuat teks menjadi rata tengah dan properti color diatur ke red untuk memberi teks warna merah. Dengan demikian, semua elemen <p> dalam dokumen akan menerapkan gaya ini.
